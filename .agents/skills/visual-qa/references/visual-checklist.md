# Visual QA Checklist

## Viewports

- Desktop: 1440px wide.
- Tablet: 768px wide.
- Mobile: 390px wide.

## Pages

- Home page.
- Collection page.
- Product detail page.
- Wishlist page.
- Checkout page.
- Account login/register/account pages.

## Interactions

- Header links and mobile menu.
- Mega menu.
- Search modal.
- Product card hover and quick view.
- Wishlist toggle.
- Cart drawer and quantity controls.
- Product image gallery.
- Product details accordion.
- Filters and sorting.

## Visual Defects

Check for layout shift, overflow, text clipping, overlapping elements, low contrast, broken images, missing focus states, incorrect spacing, and inconsistent button styling.
